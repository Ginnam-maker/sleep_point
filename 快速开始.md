# 快速开始指南

## 🚀 立即运行

### 方法一：使用 HBuilderX（推荐）

1. **打开项目**
   - 启动 HBuilderX
   - 菜单栏：文件 → 打开目录
   - 选择 `sleep_point` 文件夹

2. **运行到微信开发者工具**
   - 点击工具栏的运行按钮
   - 选择：运行 → 运行到小程序模拟器 → 微信开发者工具
   - 首次运行需要配置微信开发者工具路径

3. **配置微信开发者工具路径**（首次运行）
   - 工具 → 设置 → 运行配置
   - 找到"微信开发者工具路径"
   - Windows默认路径：`C:\Program Files (x86)\Tencent\微信web开发者工具\cli.bat`
   - Mac默认路径：`/Applications/wechatwebdevtools.app`

### 方法二：使用微信开发者工具

1. **导入项目**
   - 打开微信开发者工具
   - 点击"导入项目"
   - 项目目录：选择 `sleep_point` 文件夹
   - AppID：使用测试号即可

2. **开始调试**
   - 项目导入后会自动编译
   - 可以在模拟器中查看效果

## 🐛 常见问题排查

### 问题1：运行后白屏
**解决方案**：
- 检查控制台是否有报错
- 确认 `pages.json` 配置正确
- 尝试重新编译（Ctrl+R 或 Command+R）

### 问题2：打卡后状态不更新
**解决方案**：
- 打开调试器查看 Storage
- 检查数据是否保存成功
- 查看控制台是否有错误信息
- 尝试清除缓存后重试

### 问题3：底部导航不显示
**解决方案**：
- 确认 `pages.json` 中 tabBar 配置正确
- 检查页面路径是否正确
- 暂时移除了图标配置，只显示文字

### 问题4：HBuilderX 无法运行到微信开发者工具
**解决方案**：
1. 确认已安装微信开发者工具
2. 检查微信开发者工具路径配置
3. 确保微信开发者工具的"服务端口"已开启
   - 设置 → 安全设置 → 开启服务端口
